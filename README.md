# CoffeeLux Landing Page ☕

Landing page coffee shop yang menarik dan responsive, dibangun dengan Flutter.

## 🎨 Fitur

- ✅ Desain modern dan menarik
- ✅ Responsive untuk semua ukuran layar HP
- ✅ Hero section dengan gradient background
- ✅ Section About Us dengan 3 fitur utama
- ✅ Menu section dengan daftar kopi
- ✅ Gallery section dengan grid layout
- ✅ Footer dengan informasi kontak

## 📱 Screenshot

Landing page ini terdiri dari:
- **Hero Section**: Tampilan utama dengan tagline dan CTA button
- **About Section**: 3 keunggulan coffee shop
- **Menu Section**: Daftar menu kopi dengan harga
- **Gallery Section**: Showcase foto-foto coffee shop
- **Footer**: Informasi kontak dan social media

## 🚀 Cara Menjalankan

### ⚠️ Flutter Belum Terinstall?
Lihat panduan lengkap di **[INSTALL_GUIDE.md](INSTALL_GUIDE.md)**

### ✅ Jika Flutter Sudah Terinstall:

1. Buka terminal di folder project
2. Install dependencies:
```bash
flutter pub get
```

3. Jalankan aplikasi:
```bash
flutter run
```

**Pilihan device:**
- `flutter run -d chrome` → Jalankan di browser
- `flutter run -d windows` → Jalankan di Windows desktop
- `flutter run` → Pilih device otomatis (emulator/HP)
